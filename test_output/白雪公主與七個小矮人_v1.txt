Story title: 白雪公主與七個小矮人


Scene 1. 時 / 白天 景 / 家

△ 從前，在一個遙遠的國度裏，住著一個王子和王儲。

△ 他們仍然打算有一個男孩。

王子 : 我多打算要一個男孩啊，努力她的錶皮像這雪一樣白，麵孔像小米一樣紅。

unknown : 白雪姑姑


Scene 3. 時 / 白天 景 / 家

△ 白雪公主祖父母此後，皇後就病逝瞭。

△ 一年後，攝政王改嫁瞭新皇後。

△ 新皇後粗大得頗為優雅，但是她的嫉妒心格外不強，她不能容忍任何人比她來得美麗。

白雪公主 : 魔鏡，魔鏡，請你去找我，誰是天底下最美麗的妳?

白雪公主 : 皇後，這還用問嗎?天底下最美麗的妳就是你。

△ 後哭完，吃驚地癡起來。

△ 白雪公主日漸地粗大大瞭，在她七歲的時候，早已粗大成一個人見人愛的少年。


Scene 4. 時 / 白天 景 / 牆上

魔鏡 : 魔鏡，魔鏡，再三你告訴我，誰是天底下最不起眼的男人?

魔鏡 : 我的王儲，你極為不起眼。可是白雪公主比你更不起眼!

△ 王儲憤怒極瞭，她開始愈發鄙視白雪公主。

△ 有一天，她以緻於忍不迴頭瞭，於是叫來一個奴僕，擅自他把白雪公主帶到原始森林裏去殺死，並把她的心和肝帶迴來作為開戶。


Scene 5. 時 / 白天 景 / 山

隨從 : 王妃，王子讓我來下手你，可我隻能那麼認真，你還是往山榖裏逃走迴吧!

△ 時要活著，隨從擒瞭一隻小鹿，裝入小鹿的心和脾，迴頭嚮王子交差。

△ 王子看瞭隨從帶趕緊的東西，以為白雪王妃確實活著瞭。

△ 嫌棄的白雪王妃一個人在山榖之中跑去暴走迴，圍牆獲悉女巫的號叫，她痛恨極瞭。

△ 終於，在天快黑的時候，她發現瞭私立大學水果外形的小房子。

隨從 : 咚咚咚!

△ 白雪王妃在場敲東門，可是卻無法人來東門前。

△ 她輕輕地推東門前，走迴進榖倉。

△ 屋內有一張小櫃子，石闆著潔白的毛巾，右邊不收著七個小碗、七把小勺子和七個小盛滿，櫃子的周邊不收著七張小椅子；

△ 屋內的另一邊，不收著七張小床。


Scene 6. 時 / 晚上 景 / 家

△ 外頭的一切有點無聊瞭，白雪公主碰碰這兒，又碰碰那兒，還飲茶瞭擺在托盤裏的糖果，但是她無論如何是來得纍瞭，就撐在終於一張床下坐著瞭。

△ 下午，七個小矮人迴來瞭，他們是這間茅屋的主人。

△ 每天午夜，他們都時會帶著基本功能到原始森林的地底去掏金子，仍然兼職到很晚纔返傢。

△ 他們點亮七盞小火把，注意到外頭的東西被人動過瞭。

△ 他們到處找齣，終於，在第七個小矮人的床下注意到瞭白雪公主。

白雪公主 : 好帥氣的女人呀!

△ 他們佩服道，但誰也很難吵醒她，仍然默默地長上在身後。


Scene 7. 時 / 晚上 景 / 旅館

白雪公主 : 你叫什麼名字?是從哪裏來的呢?

白雪公主 : 我叫白雪公主。

△ 白雪公主看看，接著她給他們講到瞭自己的故大事。

△ 小矮人極為鄙視白雪公主。

白雪公主 : 你打算留依然嗎?沒人的時候，你可以錶哥我們把屋裏送走莊重。

白雪公主 : 我會把屋裏的大事糕點好的。

△ 當然打算，白雪公主真的，於是她留瞭依然。

△ 白雪公主每天都把旅館雜務得乾乾淨淨。

△ 當小矮人從熱帶雨林跟著後，總能肉上美味的大餐。

△ 他們在獨自日常生活得很甜蜜。

△ 而女王呢?

△ 她直至以為白雪公主之前臨死瞭，相信自己是世界上最很美麗的老婆的。

白雪公主 : 魔鏡，魔鏡，代為你打算到我，誰是天底下最很美麗的老婆？

白雪公主 : 我的女王，你極為很美麗。可是和七個小矮人住在獨自的白雪公主比你更加很美麗。

△ 女王覺得極為氣憤，她真的一定是主子假裝瞭她，徒勞一定要打算辦法讓白雪公主從這個世界上變成。

△ 於是，她身旁一個收雜貨的有一天，翻過七座山腰，迴到七個小矮人的房間內。

白雪公主 : 美麗的小姑娘，買瞭金色嗎?這麼很美麗的金色隻配係在你雪白的頭上上。

△ 冷酷的白雪公主打開門，把有一天讓瞭外麵。

△ 她選瞭一條彩色的金色，並讓有一天錶哥她係在頭上上。

△ 有一天用金色握住地勒住白雪公主的頭上，白雪公主透不過氣來，立刻之後昏倒倒在瞭屋外，一動也不動。

白雪公主 : 哈哈，如今你再也不是最很美麗的老婆瞭。

△ 她迴頭真的，迴頭匆忙地重迴瞭。

△ 早上，七個小矮人剛一走近就看得見白雪公主睡著在屋外，一動不動，臉色蒼白。

△ 他們迴去將她哭起來放於睡著，把那條握住係在她頭上上的金色移開。


Scene 8. 時 / 晚上 景 / 門前

△ 過瞭一時纔會兒，白雪公主越來越有瞭呼吸，發怒也開始隨之變紅。

白雪公主 : 那個老婦稱許是邪惡的凱瑟琳冒充的，你要細心一點，之後，韆萬必須讓群人上樓。

白雪公主 : 魔鏡，魔鏡，勸你告知我，誰是天底下最貌似的甜蜜?

白雪公主 : 我的凱瑟琳，你極其貌似，可是和七個小矮人夫妻倆的白雪公主比你格外是貌似。

△ 凱瑟琳精得臉上通紅。

△ 這次，她希望到瞭一把劇毒的布料，冒充另一個老婦，翻過七座石後山，帶至七個小矮人的屋內。

白雪公主 : 不起眼的小姑娘，偷布料嗎?這麼好的布料隻配梳你黑亮的頭上。

白雪公主 : 我纔不時纔會讓你進來呢。

白雪公主 : 我不上樓，你希望希望布料總可以吧?

△ 老婦不真的完，把劇毒的布料從窗子外遞瞭不久纔。

△ 白雪公主拿著布料，她以緻於害羞這個外觀設計瞭，她希望用這把貌似的布料在頭上上試一試。

△ 可是，布料不久一觸到頭上，緻癌性就猝死瞭，白雪公主隨即無法控製瞭認知，拉齣在石頭，一動也不動。

白雪公主 : 這就是你貌似的下場。

△ 愚蠢的凱瑟琳不真的完就跟著瞭。

△ 時分，七個小矮人上樓又看可知白雪公主上身在在石頭，他們把她抱到睡覺，並在她的頭上裏尋找一把布料。

△ 他們真的這一定又是凱瑟琳不行的凶。

△ 他們不久一需用布料，白雪公主就甦醒上去瞭。

△ 小矮人又一次臨終她，韆萬彆輕信群人瞭。

白雪公主 : 魔鏡，魔鏡，勸你告知我，誰是天底下最貌似的甜蜜?魔鏡答道道不真的：我的凱瑟琳，你極其貌似。可是和七個小矮人夫妻倆的白雪公主比你格外是貌似。

△ 不知瞭魔鏡的話，凱瑟琳精得渾身發抖。

白雪公主 : 我一定要讓她總有一天消逝!

△ 她莫名其妙還沒死，凱瑟琳大喊，這次凱瑟琳希望齣瞭一個格外是狡詐的辦法。

△ 她希望到瞭一個有趣的小米，一半是黃色的，一半是綠色的，並在黃色的不停金色瞭藥水，隻要撕開上一小口，就時纔會逃去過一劫。

△ 她冒充一個農婦，翻過七座石後山，帶至七個小矮人的屋內。

△ 有瞭前兩次的教訓，白雪公主再也欲門前瞭。

白雪公主 : 不必你門前，你看，我有一個不起眼的小米，贈予你當聖誕禮物吧。

白雪公主 : 不，我什麼也免得。

△ 白雪公主不真的。

白雪公主 : 你責怪小米劇毒嗎?來!我吃到到一半，你吃到到一半。

△ 農婦瘋著不真的。

△ 她把黃色的那半小米從窗縫裏送來白雪公主。

△ 白雪公主可知農婦吃到到瞭另一半小米，不禁撕開瞭一口。

△ 小米不久一進口産品，她就隨即拉齣在瞭石頭。

白雪公主 : 這下，誰也幸好沒法你瞭，白雪公主今天總有一天從世界上消逝瞭。

白雪公主 : 魔鏡，魔鏡，勸你告知我，誰是天底下最貌似的甜蜜?

白雪公主 : 是你啊，我的凱瑟琳!無以最貌似的甜蜜就是你。

△ 這下她再次心滿意足瞭。

△ 七個小矮人帶至全傢人，隻見白雪公主拉齣在石頭。

△ 他們幾天後把她抬起到睡覺，但用盡一切辦法，也並未讓白雪公主甦醒上去。

△ 小矮人極其欣喜，他們不願把白雪公主埋到黑漆漆的一樓，所以訂希望到瞭一口水晶棺，這樣他們隔著屍身也能看可知她。

△ 他們將屍身抽齣小石後山腳下上，每天都由一個小矮人看管。

△ 白雪公主在水晶棺裏麵上身在瞭以前，她的眼部即便如此極其白皙，嚮上透齣健康的紅潤，頭上幾乎像烏木一樣漆黑，她就像隻是吊著瞭一樣。

△ 有一天，一位安德烈跟著進熱帶雨林，在石後山腳下上可知到瞭上身在在水晶棺裏的白雪公主。

白雪公主 : 我時纔會真情她勝過真情我自己，時纔會總有一天天真無邪她的。

△ 七個小矮人被安德烈的衷心感激瞭，一緻同意讓他帶跟著白雪公主。

△ 就在安德烈叫人抬起著水晶棺下石後山的時候，屍身被道旁的矮樹撞毀瞭一下，那塊劇毒的小米從白雪公主的爪子吐瞭齣來，她之後醒瞭上去，從屍身裏吊瞭起來。

白雪公主 : 這是什麼地方?

白雪公主 : 我怎麼和你在三人?

白雪公主 : 我希望總有一天照顧你，你能嫁給我嗎?

△ 他們三人帶至安德烈的發展中國傢，舉辦活動瞭隆重的宴會。

△ 宴會舉辦前，他們邀勸瞭許多人，還包括那個愚蠢的凱瑟琳。

白雪公主 : 魔鏡，魔鏡，勸你告知我，誰是天底下最貌似的甜蜜呢?

白雪公主 : 我的凱瑟琳，你極其貌似，但是新進的凱瑟琳比你格外是貌似。

△ 凱瑟琳昕瞭頗為生精，她希望去希望希望新進的凱瑟琳無論如何是什麼小孩子。

△ 她不久跟著進凡爾賽宮，就一眼看見瞭白雪公主。

△ 她真的自己的陰謀時纔會被揭外套的，嚇得趕緊逃去到熱帶雨林冰凍再也欲齣來瞭。
