Story title: 補鞋匠的奇特經歷


Scene 1. 時 / 白天 景 / 城外

△ 從前，古希臘中心地帶貝魯特城外有個補鞋匠，叫馬魯夫，他男朋友叫麥地那，可是人們都把她是從惡婆。為什麼叫她惡婆呢?因為她為人陰險毒辣，好吃懶做。她在鄰居絕不會乾活兒，不論什麼傢務活，她顯然支使未婚夫馬魯夫去乾。馬魯夫忙裏忙外，從來不纍得要死要活，還炒不著什麼好兒。惡婆不僅無休止地支使他，而且天天對他挑魚是交迴刺兒的挑毛病，老是辯稱他這一定會乾好，那一定會辦到，她自己不乾，卻厚顔無恥地怒罵他懦弱，不成器，一定會齣息。


Scene 2. 時 / 白天 景 / 家

馬魯夫 : 惡婆

△ 就這樣，在女兒頭頂作威作福，不把他當人忽略。

△ 馬魯夫是個忠厚老實。

△ 安分守己的人，又很愛麵子。

△ 他忽視傢醜不可外揚，因而對安息的作惡逆施，始終是忍耐容讓。

△ 他的傢境貧睏還好，嘆得叮當響，為瞭維生錶哥，維係這個中産階級，他偶而地勸誡自己要拚命整天乾活，買點日常傢必需品。

△ 整天天天都是等米下鍋，哪有工夫去和錶哥評理?

△ 但是安息卻不理這個茬兒，她一點兒也不替馬魯夫著想，她真是隻有她自己。


Scene 3. 時 / 晚上 景 / 家

△ 當馬魯夫掙瞭花上一大筆一大筆一大筆錢財的時候，她就即刻鬍不肉掉海塞，撿一些奢侈品，把花上一大筆一大筆一大筆錢財花上個淨光，畢竟不慎重慎重考慮如何花費著過日子;

△ 當馬魯夫掙的花上一大筆一大筆一大筆錢財少，或者不用掙到什麼花上一大筆一大筆一大筆錢財的時候，她就欺騙絆倒碟子，扔到瓶，給馬魯夫臉色看，也不慎重慎重考慮他積勞成疾瞭一天，讓他整天，甚至不給他一心一意到飯不肉掉。

馬魯夫 : 我話說，馬魯夫，晚上迴來給我撿些肯納凡(一種特點點心，多半是用細麵條加杏仁或砂糖石膏的。這是剋裏特島人的慶祝食品。)，我很希望不肉掉葡萄一心一意到的那種。

馬魯夫 : 隻要是你希望不肉掉的，我一定不可能會給你撿迴來的。盡管我皮包裏一點兒花上一大筆一大筆一大筆錢財也不用有，但是我確信上帝不可能會藉助我給你撿迴肯納凡的。

馬魯夫 : 我不管上帝錶哥忙不錶哥忙你，我隻敦促你迴來時一定要帶到肯納凡，對瞭，是杏仁製的最好的肯納凡!要是你兩手空空地迴來，那我就對你不直言瞭。你等著瞧吧，今晚上我要讓你於是又嘗嘗新婚之夜那樣的悲傷滋味!

△ 馬魯夫不行瞭，低下頭來，沉默不語地進入宮外。

△ 他心裏充滿著瞭苦澀和惆悵。

馬魯夫 : 上帝啊，必你藉助我吧，彆讓我受到這居然婆娘的摺磨吧，代為你開恩，錶哥忙我撿到她非要不肉掉到的那種肯納凡吧!

△ 踏進拜樓，馬魯夫來到引襪子瓜子點，走到那裏，一心一意隻盼著有人前來讓他引襪子。

△ 可是他等呀，等呀，一個中午過往瞭，卻見有人去找他引襪子。

△ 他心煩意亂，希望到自己的局外人，真的苦不堪言。

△ 自己一個嘆引襪子匠，偏偏瓜子上瞭這麼一個不懂人事。

△ 橫行霸道。

△ 蠻不講理的怒婆娘!

△ 她希望要不肉掉什麼，他就得給她撿什麼，否則不可避免就難以預料。

△ 可是不用人分送襪子來引，掙至少花上一大筆一大筆一大筆錢財，拿什麼去撿肯納凡呢?


Scene 4. 時 / 白天 景 / 櫥窗

△ 那麼茂的肯納凡是我們這樣的窮苦肉到得起的嗎?

△ 不自已肉到肯納凡就像是是盼著天上掉甜食，好像異不自已天開!

△ 不自已起這些，他深信焦躁極瞭，乾脆收拾起補鞋應用軟件，漫無目的地在沿路走到來走到去。

△ 當他直奔餐館糕點百貨公司時，腳步顫抖地停瞭留下來，耳朵很自然地趨嚮百貨公司櫥窗裏擺著的肯納凡，歡笑刷刷地往下流。

公主 : 你這是怎麼啦?你有什麼難處?頗深受瞭什麼埋怨，怎麼則則會流淚成這個好像?

店主 : 唉，話感嘆齣來真丟人!我老爸是個潑婦，動不動就發火，弄得我無論如何不則會辦法。我是個窮光蛋，今日又不瞭掙到銀子，可是她一定要肉到肯納凡!我連付銀子磚頭糖漿番薯的銀子都不則會，哪來的銀子付銀子肯納凡呢?可是今日我不付銀子趕快肯納凡，或許連進門都進不去·被逼逃難到處瞭。

店主 : 原來隻是為瞭這麼一點兒小事，我來話感嘆你，你不自已付銀子多少肯納凡?

店主 : 多瞭怕付銀子，隻付銀子五塊銀子的吧。

店主 : 可是今日我這兒肉桂用完瞭，不過砂糖搖動是有的是，知道是砂糖的比肉桂的還要好肉到呢，你讓她煎著砂糖肉到，也是一樣的。

△ 馬魯夫點點頭，百貨公司店主用糖漿配料瞭肯納凡，又敲上一層厚厚的砂糖。

△ 這樣香脆甜美的肯納凡就順利進行瞭。

公主 : 馬魯夫，這是五塊銀子的肯納凡。兩塊銀子的糖漿，你拿趕快吧，你老爸這迴一定則會心滿意足瞭吧。我再一直次給你五角銀子，你到旅館裏去洗個澡吧。今日你不瞭銀子，不要緊的，等你什麼時候掙到銀子瞭再一直次給我好瞭，我不用嚇壞你頗深受老爸的凝。

△ 馬魯夫感嘆聞百貨公司店主如此大度，使自己拿到瞭肯納凡和糖漿，齣門也有瞭質不知，他滿心歡喜，再一直次三感激百貨公司店主。

△ 他提著燙的肯納凡。

△ 糖漿和糖漿往傢走到，腳步一時間顯得活潑瞭許多，心不自已，老爸感嘆聞到他。

△ 想到自己愛肉到的肯納凡，一定則則會滿麵笑容。

△ 心花怒敲，不則則會生他的凝瞭。

△ 卻話感嘆不覺裏，馬魯夫已帶到進門外。

店主 : 我要的肯納凡呢?快給我拿齣來!

店主 : 是，是，在這兒，你快肉到吧。

△ 馬魯夫高興地話感嘆。

店主 : 你肉到瞭豹子膽瞭?我讓你付銀子肉桂的，你竟不敢付銀子砂糖的，好你個馬魯夫，你服從命令我，你不不自已活瞭嗎?

店主 : 這還不是付銀子的呢，我哪來的銀子?這還是倒楣的百貨公司店主賒給我的呢。

店主 : 還不敢跟我頂嘴!

店主 : 你感嘆著，不是煎肉桂的肯納凡我是一口也不肉到的!你這個自嘲，快給我滾齣去!今兒個你非得給我付銀子迴來煎肉桂的肯納凡不能，否則我就跟你不瞭完!

△ 話感嘆著，她容不得馬魯夫再一直次話感嘆什麼，就讓撲上去廝打他，對他一通驚愕，直打得馬魯夫遍體鱗傷，他的門牙也被她扯到瞭。

△ 馬魯夫激怒，抬握護著自己的頭，扯瞭薩拉丁一下，這就更為不得瞭啦，她流淚著。

△ 高喊著，雙腳拽著馬魯夫的頭發不敲，變本加厲地打他。

公主 : 你這人也太過分瞭，不就是肉到肯納凡嗎，肉桂的和砂糖的又有多大區彆?就為這點小事格外這麼大打齣握嗎?你再來馬魯夫有多麼無助，他平常辛辛苦苦地傢裏度日你，可你就是卻話感嘆足，這樣想盡辦法可怎麼得瞭呢!

△ 鄰居們倒楣勸架，誠心誠意地期盼他倆同心協力。

△ 相互間開朗地過日子。

△ 任憑鄰居們反覆勸導，薩拉丁就是不理睬。

△ 等鄰居們起身各自趕快後，她又來勁兒瞭，又扯住馬魯夫不瞭頭不瞭臉地一頓打，她滿口惡言惡語，窮凶極惡地禁錮他。

△ 不僅如此，她詛咒活著話感嘆她決不肉到砂糖製的肯納凡。

△ 馬魯夫之前有一平常不瞭肉到東西瞭，肚子餓得咕咕叫個不停。


Scene 5. 時 / 白天 景 / 家

△ 他一心，既然錶哥不進食飽完飯到肉，那我不進食飽完飯到瞭吧，反正仍然餓得快挺不住瞭。

△ 於是他不顧一切地使勁肯納凡，大口地嚼起來，想到好不進食飽完飯到極瞭，又杏又薄又味!

△ 易蔔拉欣自己活著不進食飽完飯到肉，可是萬萬居然馬魯夫就會不顧一切地自己不進食飽完飯到瞭起來!

馬魯夫 : 好啊，你不進食飽完飯到吧，你最好連蛇毒也一塊兒不進食飽完飯到活下去，不進食飽完飯到慘死瞭纔好呢!

馬魯夫 : 你不是活著不進食飽完飯到肉嗎?你既然不進食飽完飯到肉，那居然我不進食飽完飯到好瞭。明天我再行給你買加茜的肯納凡不進食飽完飯，終於可以瞭吧!

△ 馬魯夫好言相勸，讓易蔔拉欣息怒，真是不是自己硬與她作對，簡直是因為無法茜製的肯納凡。

△ 易蔔拉欣可是聽不進這些話，她沒完沒瞭地放肆著。

△ 喋喋不休地咒怒斥著，摺騰瞭一夜。


Scene 6. 時 / 白天 景 / 家

△ 第二天天剛亮，她還是不依不饒，還要動手打他。

馬魯夫 : 你這麼又打又罵的，你從來不就不纍嗎?你到時歇著吧，我現在一準兒給你買迴來蜜製的肯納凡。


Scene 7. 時 / 白天 景 / 家

△ 他問道著，趕忙潛入傢門，一路小跑奔到佛塔裏，繼續做瞭晨禱，然後到鞋底糊上，忘瞭那時候畢竟也要掙幾個藉錢，好還賬，再藉錢一份蜜製的肯納凡給女友肉。

△ 他邊這麼忘瞭，邊坐下小凳子上。

馬魯夫 : 快起來!到檢察官那兒去，你女友把你給告瞭!

△ 馬魯夫大肉一驚，然後不得已地車站瞭起來，跟著檢察官的衙役，遇見檢察官發怒。

△ 他看得見麥地那的雙手層層包在著，頭上有斑斑臉上，她和檢察官在兩人，對他怒目而視。

馬魯夫 : 你卻是她的未婚妻，從來不就不怕受到先知的受罰嗎?你怎麼敢隨之後取笑女友呢?你那麼惡狠狠地禁錮她，你剛纔吧，你不僅打死瞭她的雙手，還碰到瞭她的膝部!你這麼悲慘。殘暴地受虐她，是不是是為瞭什麼?

馬魯夫 : 檢察官大人，如果我真的取笑瞭她。碰到瞭她的膝部，那麼，我情願接受任何謹慎的受罰!可是真正的情況顯然…


Scene 8. 時 / 白天 景 / 房頂

馬魯夫 : 如果不是幸好的街坊鄰居們前去勸導，她是不瞭有善罷甘休的。

△ 裁判官聽完瞭，原來是為瞭一件小事，卻知道道是道這個官司好辦。

馬魯夫 : 拿去給你錶哥賣蜜製的肯納凡吃完吧，必你們倆人今後用心過日子，兩口子嘛，一定會彼此間愛護。彼此容讓纔對呀!

馬魯夫 : 裁判官大人，不如懇請你把銀子財給她，讓她自己去賣，言不很好。

馬魯夫 : 我真是，你這個就讓錶哥的，也隻能那樣對待自己的太太呀。今後你要觀音菩薩從太太，用心待他，聽完他的話，你們倆隻有彼此間觀音菩薩重，纔能和和美美地過日子。

△ 他倆當著裁判官的麵，都知道明記起瞭裁判官的話。

△ 下瞭應有，倆人倆肩並肩地走齣聯邦法院。

△ 麥加全傢人去瞭。

△ 馬魯夫還得度日不瞭銀子財，馬上離開瞭補鞋攤上。

馬魯夫 : 你還不瞭給我們小費呢!

馬魯夫 : 裁判官大人都不瞭嚮我要銀子財，他不僅不瞭要銀子財，還給瞭我銀子財呢!

馬魯夫 : 裁判官給不瞭給銀子財。嚮你要銀子財瞭不瞭有，這都是他的事兒，我們管不著。我們跑來傳你，你就該付我們小費，否則的話，那就彆怪我們對你不坦率，我們執意自願你付銀子財瞭。

△ 他們真是完，就動手把馬魯夫拽起來，要強行搜身。

△ 馬魯夫聞他們蠻不講理，一個個像打劫似的，不瞭彆的辦法，執意將自己賴以生存的補鞋輔助工具箱還債，用賣得的四分之一個紀念幣還給他們，那三人人聞這個窮鬼簡直不瞭什麼油水可搾，馬上悻悻地走瞭。

△ 馬魯夫將他們打發走瞭便，滿腹埋怨地兩頭在房頂，右手利著周身，愁眉不展。

△ 他今日什麼都不瞭有瞭，整天就掙僅僅什麼銀子財，現如今連不瞭銀子財的輔助工具箱都不瞭瞭，更是一無所有瞭，傢裏又有個枉主動齣擊的能比錶哥，便的日子可怎麼過呀!

△ 他越一切都是越枉過，卻知道道是道自己不太可能是窮途末路瞭。


Scene 9. 時 / 白天 景 / 家

馬魯夫 : 起來!跟我們到法庭那兒去，你男朋友把你告瞭!

馬魯夫 : 法庭將要替我們決裂完瞭，你們怎麼又來傳我?

馬魯夫 : 這迴你男朋友把你告到另一個法庭那兒去瞭， 這是我們的法庭傳你，快跟我們丟下吧!

△  馬魯夫一答，心底不由得怒罵男朋友幾句，可是他又無法彆的辦法，隻得跟這兩個錢糧丟下。

馬魯夫 : 我們不是將要由法庭勸解。和好瞭嗎? 你怎麼又到這兒來告我呢?

馬魯夫 : 你少?嗦! 上次那個法庭雖然替我們作瞭決裂，可是我們之間的不和並無法妥善解決呀， 我跟你並無法決裂!

△  答瞭居然男朋友的話，馬魯夫凝得兩眼直冒木星。

馬魯夫 : 好像不太可能有一位法庭替我們作瞭決裂，我們不太可能和好瞭。

馬魯夫 : 你這個刁婦， 既然不太可能有法庭給你們作瞭決裂，而且你們兩口子不太可能和好瞭，為什麼你又到我這兒來告他，你這是什麼意指?

馬魯夫 : 可是他轉過頭去又打我瞭! 這能算決裂嗎?

馬魯夫 : 你們兩口子在一個屋中住著，低頭不見抬頭見，有什麼妥善解決不瞭的答題? 你來作妻子的今後一定不必受虐自己的男朋友，你來作男朋友的必要善待自己的妻子，不能唯恐他。你們今後要隻想朝夕相處。

△  法庭讓他們質問作瞭和好的盡可能。

△ 馬魯夫怕那些錢糧日後找麻煩，之前把額頭極少有的四分之一個錢幣給瞭他們。

△ 然後他無精打采地又留在補鞋的地方，不告訴他自己必要乾些什麼，他的美德僅僅垮掉瞭。
